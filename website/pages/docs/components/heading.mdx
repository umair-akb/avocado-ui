# Heading

The Heading component extends the specified `h*` HTML element. You'll use it to render different levels of headings.

## Usage

Import the `Heading` component from `@avocado-ui/react` and render like so

```jsx live noInline
import { Heading } from '@avocado-ui/react'
render(<Heading level='h1'>Heading</Heading>)
```

### Level

There're 6 heading levels in HTML. You can specify the preferred heading with the `level` prop like so

```jsx live noInline
import { Heading } from '@avocado-ui/react'
render(
  <>
    <Heading level='h1'>Heading 1</Heading>
    <Heading level='h2'>Heading 2</Heading>
    <Heading level='h3'>Heading 3</Heading>
    <Heading level='h4'>Heading 4</Heading>
    <Heading level='h5'>Heading 5</Heading>
    <Heading level='h6'>Heading 6</Heading>
  </>
)
```

### Size

You can also override the size for different heading levels with the `size` prop. The supported values are as follows `xs`, `ms`, `sm`, `lg`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl`, `7xl`.

```jsx live noInline
import { Heading } from '@avocado-ui/react'

render(
  <>
    <Heading level='h1' size='xs'>
      Heading 1
    </Heading>
    <Heading level='h2' size='ms'>
      Heading 2
    </Heading>
    <Heading level='h3' size='sm'>
      Heading 3
    </Heading>
    <Heading level='h4' size='lg'>
      Heading 4
    </Heading>
    <Heading level='h5' size='lg'>
      Heading 5
    </Heading>
    <Heading level='h6' size='xl'>
      Heading 6
    </Heading>
  </>
)
```

## Heading Props

| property | description                   | type                                                                   | default |
| -------- | ----------------------------- | ---------------------------------------------------------------------- | ------- |
| level    | level of heading              | `h1`, `h2`,`h3`                                                        | null    |
| size     | size to override heading size | `xs`, `ms`, `sm`, `lg`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl`, `7xl` | `level` |
